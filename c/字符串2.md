### 字符串指针表示
```c
#include <stdio.h>

int main() {
    char * str = "hello world!";
    printf("%s\n", str);
    return 0;
}
```

"h" 与 'h' 区别:
1. 长度: "h"占用`2`字节(包含尾零'\0'), 'h'占用`4`字节.

### 当我们写"hello"时,就是告诉编译器,在`只读数据段`存储6个字符(注意包含尾零);
### "hello"表达式的值表示此字符串的第一个字符的地址,即字符h的地址,在字符串中,字符是占用1个字节

字符串经典示例,说明字符串不是存储在函数中的栈里,而是存储在另外的内存
```c
#include <stdio.h>

void func(void) {
    char * p = "hello"; //字符串保存在只读数据段,不是保存在当前函数的执行栈中
    write(1, p, 10); // 从p地址开始,强制写入10字节到屏幕上
    return;
}

int main(){
    char * q = "world\n"; //字符串保存在只读数据段,不是保存在当前函数的执行栈中
    func();
    
    // 获取首地址的字符
    printf("%c\n", *"hello"); // h

    // 注意只读数据是不能修改的
    char * p = "hello"; 直接操作只读区数据
    p[0] = 'H'; //会报错的

    // 但是,
    char a[] = "hello"; //注意这个表达式: 首先将字符串"hello"保存到只读数据区, 然后再将其复制到函数栈中的数组中
    printf("%s", a);
    a[0] = 'H'; //这里的a[0],是可以修改的,操作的是函数栈中的数据
    printf("%s", a);


    return 0;
}
```
